# Quadratics-Solver

--Description--
Solves quadratics with a user chosen method given a few options. Use in a terminal. This program was created long before the creation of this repository, so this project is just being moved to GitHub.
This program was created as an academic project to help kids learn how to solve quadratic equations using various methods through a step by step process.
This project was started and completed during May, 2025.
